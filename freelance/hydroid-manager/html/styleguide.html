Hydroid Manager
Style guidelines

This document contains a list of guidelines which describe how the Hydroid Manager app should appear. Its purpose is to promote aesthetic consistency during the application's development. These provisions may be modified or updated in the future.

Spacing

Spacing and font sizes are defined using rem, which calculates sizes relative to the font size of the root HTML element. In this case all spacings are defined relative to 16px.
